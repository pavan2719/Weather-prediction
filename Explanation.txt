This project is an interactive weather prediction system built using Python. It predicts the next day’s temperature based on past data using a simple machine learning model. The model used is Linear Regression from scikit-learn, which learns the relationship between days and temperature values. The user enters past temperature data through a web interface created using Streamlit. This input is processed using NumPy to convert it into a structured format suitable for the model.

After training, the model predicts the temperature for the next day. The result is displayed instantly on the screen. Additionally, the project uses Plotly to create an interactive graph that shows both past data and the predicted value.The graph allows users to zoom, hover, and explore trends, making the output more understandable. This project demonstrates the basic concept of machine learning, data visualization, and user interaction in a simple and effective way.
                                                     
                                                         It is useful as a beginner-level data science project and can be extended using real-time weather APIs or advanced models like time-series forecasting.
